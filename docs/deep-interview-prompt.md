## 🚀 활용 가이드 및 팁

### 1. 사용 환경

* **Claude Code**의 `Plan Mode`와 함께 사용하면 더욱 효과적입니다.
* 작성된 `SPEC.md`나 기획 초안이 있을 때 실행하세요.

### 2. 인터뷰의 깊이

* 단순한 기능 구현 질문에 그치지 않고, **15개에서 많게는 60개 이상의 질문**을 주고받으며 설계를 다듬을 수 있습니다.
* 질문 답변 과정 자체가 문서화(Documentation) 과정이 됩니다.

### 3. 기대 효과

* **전략 정렬:** 개발 시작 전, 사용자가 구상한 그림과 Claude의 이해도를 일치시킵니다.
* **엣지 케이스 발견:** 미처 생각하지 못한 UI/UX 예외 상황이나 기술적 제약 사항을 미리 파악합니다.
* **정확도 향상:** 심층 인터뷰 후 생성된 코드는 수정 수정 횟수를 획기적으로 줄여줍니다.
