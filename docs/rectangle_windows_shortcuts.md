# Rectangle (Windows) Keyboard Shortcuts Cheat Sheet

This document lists the keyboard shortcuts for **RectangleWin**, the Windows port.
These have been adapted to avoid conflicts with native Windows system shortcuts.

## Halves & Maximize
| Action | Shortcut |
| :--- | :--- |
| **Left Half** | CTRL + ALT + LEFT_ARROW |
| **Right Half** | CTRL + ALT + RIGHT_ARROW |
| **Top Half** | CTRL + ALT + UP_ARROW |
| **Bottom Half** | CTRL + ALT + DOWN_ARROW |
| **Maximize** | CTRL + ALT + ENTER |
| **Center** | CTRL + ALT + C |
| **Restore** | CTRL + ALT + BACKSPACE |

## Corners (Quadrants)
| Action | Shortcut |
| :--- | :--- |
| **Top Left** | CTRL + ALT + U |
| **Top Right** | CTRL + ALT + I |
| **Bottom Left** | CTRL + ALT + J |
| **Bottom Right** | CTRL + ALT + K |

## Thirds
| Action | Shortcut |
| :--- | :--- |
| **First Third (Left)** | CTRL + ALT + D |
| **Center Third** | CTRL + ALT + F |
| **Last Third (Right)** | CTRL + ALT + G |
| **First Two Thirds** | CTRL + ALT + E |
| **Last Two Thirds** | CTRL + ALT + T |

## Size
| Action | Shortcut |
| :--- | :--- |
| **Make Smaller** | CTRL + ALT + - |
| **Make Larger** | CTRL + ALT + + |

## Multi-Display
Repeatedly pressing the same shortcut (Halves, Thirds) will move the window to the next display and wrap around
