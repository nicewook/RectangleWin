# Rectangle (macOS) Keyboard Shortcuts Cheat Sheet

This document lists the default keyboard shortcuts for the [Rectangle](https://rectangleapp.com/) app on macOS.
Rectangle commonly uses the **CTRL + OPT** combination as the base modifier.

## Halves & Maximize
| Action | Shortcut |
| :--- | :--- |
| **Left Half** | CTRL + OPT + LEFT_ARROW |
| **Right Half** | CTRL + OPT + RIGHT_ARROW |
| **Top Half** | CTRL + OPT + UP_ARROW |
| **Bottom Half** | CTRL + OPT + DOWN_ARROW |
| **Maximize** | CTRL + OPT + ENTER |
| **Center** | CTRL + OPT + C |
| **Restore** | CTRL + OPT + DELETE |

## Corners (Quadrants)
| Action | Shortcut |
| :--- | :--- |
| **Top Left** | CTRL + OPT + U |
| **Top Right** | CTRL + OPT + I |
| **Bottom Left** | CTRL + OPT + J |
| **Bottom Right** | CTRL + OPT + K |

## Thirds
| Action | Shortcut |
| :--- | :--- |
| **First Third (Left)** | CTRL + OPT + D |
| **Center Third** | CTRL + OPT + F |
| **Last Third (Right)** | CTRL + OPT + G |
| **First Two Thirds** | CTRL + OPT + E |
| **Last Two Thirds** | CTRL + OPT + T |

## Size & Move
| Action | Shortcut |
| :--- | :--- |
| **Make Smaller** | CTRL + OPT + - |
| **Make Larger** | CTRL + OPT + + |
| **Move Left** | CTRL + CMD + LEFT_ARROW |
| **Move Right** | CTRL + CMD + RIGHT_ARROW |
| **Move Up** | CTRL + CMD + UP_ARROW |
| **Move Down** | CTRL + CMD + DOWN_ARROW |

## Multi-Display
| Action | Shortcut |
| :--- | :--- |
| **Next Display** | CTRL + OPT + CMD + RIGHT_ARROW |
| **Previous Display** | CTRL + OPT + CMD + LEFT_ARROW |

## Other
| Action | Shortcut |
| :--- | :--- |
| **Maximize Height** | CTRL + OPT + SHIFT + UP_ARROW |
| **Almost Maximize** | CTRL + OPT + SHIFT + ENTER |
| **Pin / Unpin** | (User Configurable) |
